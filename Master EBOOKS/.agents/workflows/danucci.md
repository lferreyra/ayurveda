# Proyecto DANUCCI - Workflow de Desarrollo

Este workflow permite la ejecución automática de todos los comandos relacionados con el proyecto DANUCCI.

## Estructura del Proyecto
- Directorio base: `c:/Users/Usuario/Master EBOOKS/Master EBOOKS/DANUCCI/`
- Todos los archivos del proyecto deben vivir dentro de esta carpeta.

## Pasos de Trabajo

1. Crear archivos HTML, CSS y JS dentro de la carpeta DANUCCI.
2. Mover o copiar assets necesarios.
3. Organizar subcarpetas (assets, images, etc).
4. Ejecutar servidores de desarrollo locales.
5. Instalar dependencias si se requieren.
